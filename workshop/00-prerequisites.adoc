== Prerequisites

As prerequisites to perform the workshop, you need to have the following tools installed on your laptop:

- JDK 8+
- Maven 3
- Docker
- `kubectl`
- https://cloud.ibm.com/docs/home/tools[IBM Cloud CLI^] (`ibmcloud`)
